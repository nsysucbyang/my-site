BLAST
BLAST為在NCBI網佔所提供資料庫搜尋的工具，主要在做序列的比對。
除提供多資料庫快速查詢功能外亦同時提供數種重要統計方法供研究使用。
The NCBI PubMed PubMed是美國國家醫學圖書館提供使用者免費檢索的電子資料庫，可以免費查詢文獻 摘要。檢索結果若有全文，則可直接連結到出版社的網站，大部份皆必須付費訂購全文。
ExPASy 蛋白質資料庫 -- ExPASy Molecular Biology Server
ExPASy為全世界最重要的蛋白質資料庫，目前為GCG最主要的蛋白質資料來源。
主要內容包括蛋白質序列、構造的多個重要資料庫以及收集蛋白質序列和構造的分析工具。
The Genome Database(GDB)基因體資料庫鏡相站 GDB是人類的Genome資料庫，主要存放人類基因體的資料，收錄了人類基因體所有的圖譜資訊。
The NCBI Entrez Taxonomy Homepage
將已知的含有核甘酸和蛋白質的有機體按分類法排好，可以選擇想看的分支之後再加以觀看。
Fly Base 這是一個介紹有關目前對果蠅研究的網站，包含果蠅的DNA序列，一些reference到果蠅的Paper。
EMBL(European Bioinformatics Institute) 一個搜尋nucleotide sequces，Protein sequence，Protien structure 的網站。
NCBI Entrez Entrez是NCBI成立的一個生化綜合搜尋系統，主要是聯結了很多相關資料庫，可查得相關論文、 DNA序列、蛋白質、三級結構、染色體、物種分類、基因表現等資訊。
MAP - Multiple Global Alignment of Sequences Using Iterative Pairwise Method 提供DNA序列比對的網站，依照其規定的型式輸入即可，有許多使用上貼心的服務，且可挑選自已喜歡的比對方法。
The NCBI OMIMTM - Online Mendelian Inheritance in ManTM 「人類線上的孟德爾」，顧名思義，他匯整了人類從以前到現在的基因相關資訊，可能來自沒分類或不知名作者的寶貴資訊或文章，可用關鍵字搜尋。
NCBI GenBank NCBI的GenBank是一個主要的基因資料庫, 它資料其中一個重要來源就是研究者直接提交的新資料.
提交的方法有用E-mail傳, 用磁片寄和用其專用軟體如BankIt和Sequin直接透過網 路上傳.

EBI ClustalW
 ClustalW是一種用來作Multi Sequences Alignment的演算法.
這個網頁是European Bioinformatics Institute以WWW介面提供人使用的.
以下網址有提供平行計算ClustalW可以在自家的工作站上用
http://www.sgi.com/solutions/sciences/chembio/resources/clustalw/parallel_clustalw.html

Bowie

同合作建構的網站.
其收集了許多資源, 有提供許多生物資訊的介紹性的資料等.
