資料結構上機習題繳交方式

(Submission of Assignments of Data Structures, See below for English version)

　

  • 程式的註解：
    1.每個程式需有下列三行註解：
       Author：作者姓名（含學號）
       Date：撰寫日期 （如 Sept. 22, 2025）
       Purpose：本程式之功能與用途
    2.每一個變數名稱以及每一個函數的參數均需加上註解
    3.在適當處加入註解（程式行數與註解行數約為2:1）

  • 繳交上機習題方式：
    1.準備下述三個檔案：
       (1) 原始程式檔（檔案名稱為 hw1-b133040099.c 或 hw1-b133040099.cpp，名稱中，前段為作業編號，後段為學號）
       (2) 執行檔（檔案名稱為 hw1-b133040099.exe）
       (3) 執行結果（檔案名稱為 hw1-b133040099.out）
    2.將上述三個檔案 上傳至      中山大學的｢網路大學｣

  • 繳交期限：
     1.每個上機習題之繳交期限均在課程網頁公布，繳交期限為到期日之23:59(23時59分），此時間將於作業繳交系統設定。
    2. 繳交作業時，若發生問題，請立即透過e-mail向助教反應。

  • 問題反應：
     e-mail 至 nsysudsproblem@gmail.com  ，標題(主旨)格式為"資結問題-問題標題"。
    例如："資結問題-關於linked-list" 。

    注意：為過濾垃圾信件，助教的信箱有設定過濾條件    。請務必依照格式
     反應問題，否則信件可能會被系統自動過濾掉而導致助教沒收到信 。

Submission of Assignments of Data Structures (English version)

  • Comments in programs:
    1.     Each program should have the following three lines of comments:
           Author: name of the author (including student ID)
       Date    : date for program design  (E.g. Sept. 22, 2025)
       Purpose    : the function  of the program
    2.    Add comments to each variable name and each of the function’s parameters.
    3.    Add comments appropriately. (The ratio between the number of program lines and the number of comment lines is approximately 2:1.)

  • Submission way of the assignments:
    1.    Prepare the following three files：
       (1)     Source program file (The file name is hw1-b133040099.c or hw1-b133040099.cpp, where the former is the assignment number, and the latter is student ID)
       (2)     Executable file (The file name is hw1-b133040099.exe)
       (3)     Output results of program execution (The file name is hw1-b133040099.out)
    2.Upload the above three files to the National Sun Yat-sen Cyber University.

  • Deadline:
     1. The due date of each assignment is published on the course web site, precisely 23:59 of the due date.
    2. If there is any problem, please contact TA      immediately via e-mails.

  • Question report:
     e-mail to  nsysudsproblem@gmail.com ,  with Title (Subject) format " Problem of data structures - question".
     e.g. "Problems of data structures-linked-list"

     Notice: Be sure to be in such format. Otherwise, the email may be automatically filtered out and TA will not receive it.
